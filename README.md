# C-Map

CMap is a simple yet fast Map Structure implementation in pure C

# Features

- Simple, and Fast
- One Header File
- Auto Re-allocation
- CRUD-Like Library (Create,Read,Update,Delete)
- Written in about 120 lines of code

# How to use
Read the `main.c` file
